# Boas práticas na programação


[] Tipo de dado: Boolean (true | false)
[] if/else (elseif)
[] Operadores comparativos
  - > (maior que)
  - < (menor que)
  - >= (maior ou igual que)
  - <= (menor ou igual a)
  - == (igual a)
  - != (diferente de)
